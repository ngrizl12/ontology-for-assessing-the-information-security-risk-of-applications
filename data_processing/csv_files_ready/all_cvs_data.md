# CSV файлы с данными

## Описание
Готовые CSV файлы с обработанными данными CVE, CPE, CWE и CAPEC.

## Файлы
- `cve_all.csv` — все CVE записи из NVD API
- `cve_all_done.csv` — обработанные CVE с сопоставленными CPE
- `cpe_all.csv` — все CPE записи из NVD API
- `cpe_in_cve.csv` — CPE, которые встречаются в CVE
- `cwe_all.csv` — все CWE записи из XML
- `capec_all.csv` — все CAPEC записи из XML
