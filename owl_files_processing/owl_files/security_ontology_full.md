# OWL онтология безопасности

## Описание
Файл OWL онтологии для представления данных о уязвимостях безопасности.

## Файлы
- `security_ontology_full.owl` — полная онтология безопасности с импортированными данными:
  - CVE (уязвимости)
  - CPE (компоненты)
  - CWE (слабости)
  - CAPEC (атаки)
  - Связи между сущностями
